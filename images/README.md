# 画像ファイル配置ガイド

## 素材写真

素材は `C:\dev\shougi_hp\img\` に配置済みです。HTMLからは、トップページでは `img/ファイル名`、下層ページでは `../img/ファイル名` の相対パスで参照します。

## 元サイト画像の追加候補

まだ取得していない画像がある場合は、以下のような名前で `img/` に保存すると管理しやすくなります。

- `gallery-trophy.jpg`: 歴代優勝者トロフィー
- `gallery-scores.jpg`: 大会名人位成績表
- `gallery-01.jpg`, `gallery-02.jpg`: 活動写真の連番

## 注意

元画像や参照用画像は `img/BackUp/` に退避しています。活動案内では、お休み情報をテキストとして転記し、掲示板風カードとして `activity.html` に実装しています。
